# Практическая работа №6 – Алгоритмы

## Задача 1
### Линейный алгоритм «Перевод времени»

Дано количество часов. Составьте алгоритм, который переводит это значение в минуты.

### Алгоритм
1. Ввести `количество часов (hours)`
2. Если `hours <= 0`, то вывести сообщение `"Количество часов должно быть больше 0"` и завершить алгоритм
3. Вычислить количество минут: `minutes = hours * 60`
4. Вывести `minutes`

### Пример работы
| Ввод | Вывод |
|------|-------|
| 2 | 120 |
| 4 | 240 |
| 0 | "Количество часов должно быть больше 0" |
| -5 | "Количество часов должно быть больше 0" |

## Задача 2
### Алгоритм ветвления «Сравнение чисел»

Даны два числа A и B. Составьте алгоритм, который сравнивает эти числа и выводит сообщение о том, какое из них больше.

### Алгоритм
1. Ввести числа `A` и `B`
2. Если `A = B`, то вывести `"Числа равны"` и завершить алгоритм
3. Если `A > B`, то вывести `"Число A больше"`, иначе вывести `"Число B больше"`

### Пример работы
| Ввод | Вывод |
|------|-------|
| A=2, B=4 | "Число B больше" |
| A=5, B=3 | "Число A больше" |
| A=1, B=1 | "Числа равны" |

## Задача 3
### Алгоритм ветвления «Допуск к экзамену»

У студента есть текущая оценка за семестр. Составьте алгоритм, который определяет, допущен ли студент к экзамену. Студент допускается, если его оценка не ниже 3.

### Алгоритм
1. Ввести `текущую оценку за семестр (avgScore)`
2. Если `avgScore < 1 или avgScore > 5`, то вывести `"Аномальная текущая оценка за семестр!"` и завершить алгоритм
3. Если `avgScore >= 3`, то вывести `"Допущен"`, иначе вывести `"Не допущен"`

### Пример работы
| Ввод | Вывод |
|------|-------|
| 5 | "Допущен" |
| 3 | "Допущен" |
| 2 | "Не допущен" |
| 6 | "Аномальная текущая оценка за семестр!" |

## Задача 4
### Алгоритм с ветвлением «Скидка в магазине»

Даны цена за единицу товара и количество купленных единиц. Если общая сумма превышает 1000 рублей — скидка 10%, иначе скидки нет.

### Алгоритм
1. Ввести `цену за единицу товара (price)` и `количество (count)`
2. Если `price <= 0 или count <= 0`, то вывести `"Некорректные данные для покупки!"` и завершить алгоритм
3. Вычислить общую сумму: `total = price * count`
4. Если `total > 1000`, то `finalTotal = total - total * 0.10`, иначе `finalTotal = total`
5. Вывести `finalTotal`

### Пример работы
| Ввод | Сумма | Скидка | Вывод |
|------|-------|--------|-------|
| price=200, count=6 | 1200 | 10% | 1080 |
| price=100, count=5 | 500 | нет | 500 |
| price=250, count=4 | 1000 | нет | 1000 |
| price=-100, count=5 | — | — | "Некорректные данные для покупки!" |

## Задача 5
### Алгоритм с вложенными ветвлениями «Квадратное уравнение»

Даны три числа: `a`, `b`, `c`. Составьте алгоритм для решения уравнения вида `ax² + bx + c = 0`.

### Алгоритм
1. Ввести `a`, `b`, `c`
2. Если `a = 0`, то вывести `"Уравнение не является квадратным"` и завершить алгоритм
3. Вычислить дискриминант: `D = b² - 4ac`
4. Если `D > 0`, то:
   - `x1 = (-b + √D) / (2a)`
   - `x2 = (-b - √D) / (2a)`
   - Вывести `"Два корня: x1, x2"`
5. Если `D = 0`, то:
   - `x = -b / (2a)`
   - Вывести `"Один корень: x"`
6. Если `D < 0`, то вывести `"Корней нет"`

### Пример работы
| Ввод | Вывод |
|------|-------|
| 1, -4, 4 | "Один корень: 2" |
| 1, 1, 1 | "Корней нет" |
| 0, 2, 5 | "Уравнение не является квадратным" |
| 1, -3, 2 | "Два корня: 2 и 1" |
