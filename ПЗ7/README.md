## Задача 1. Перевод часов в минуты

**1. Назначение программы:**
Программа нужна для того, чтобы переводить время из часов в минуты.

**2. Входные данные:**
* Число — целое или дробное.
* Вводить можно от 0 до 1000 часов.
* Вводится с клавиатуры или через интерфейс.

**3. Выходные данные:**
* Целое число — количество минут.
* Выводится просто цифрой, без лишних слов (например, 120).

**4. Как работает программа:**
1. Пользователь вводит количество часов (h).
2. Программа умножает это число на 60: m = h * 60.
3. Результат выводится на экран.

**5. Что делать при ошибках:**
* Если введено отрицательное число — написать: «Ошибка: время не может быть отрицательным».
* Если введено что-то нечисловое — написать: «Ошибка: введите корректное число».

**6. Интерфейс:**
* Поле для ввода часов.
* Кнопка «Перевести».
* Место, где отображается результат.

## Задача 2. Сравнение двух чисел

**1. Назначение программы:**
Программа берёт два числа и говорит, какое из них больше — или что они равны.

**2. Входные данные:**
* Два числа — целые или дробные.
* Допустимый диапазон: от -10 000 до 10 000.
* Каждое число вводится в отдельное поле.

**3. Выходные данные:**
* Одна из трёх фраз:
    * «Первое число больше второго»,
    * «Второе число больше первого»,
    * «Числа равны».

**4. Как работает программа:**
1. Пользователь вводит первое число (a) и второе (b).
2. Программа их сравнивает:
    * a > b → «Первое число больше второго».
    * a < b → «Второе число больше первого».
    * a = b → «Числа равны».

**5. Что делать при ошибках:**
* Если введено не число — написать: «Ошибка: введите числа».

**6. Интерфейс:**
* Два поля для ввода.
* Кнопка «Сравнить».
* Место для вывода результата.


## Задача 3. Допуск к экзамену

**1. Назначение программы:**
Программа проверяет, может ли студент сдавать экзамен — на основе посещаемости и выполненных заданий.

**2. Входные данные:**
* Посещаемость в процентах: от 0 до 100.
* Процент выполненных заданий: от 0 до 100.

**3. Условия допуска:**
* Посещаемость не меньше 75%.
* Заданий выполнено не меньше 80%.
* Если оба условия выполнены — студент допущен.

**4. Выходные данные:**
* «Допущен к экзамену» или «Не допущен к экзамену».

**5. Как работает программа:**
1. Ввести посещаемость (p) и процент заданий (z).
2. Если p >= 75 и z >= 80 — вывести «Допущен к экзамену».
3. Иначе — «Не допущен к экзамену».

**6. Что делать при ошибках:**
* Значение вне диапазона [0; 100] — «Ошибка: введите значения от 0 до 100».
* Не число — «Ошибка: введите числа».

**7. Интерфейс:**
* Поле для посещаемости.
* Поле для процента заданий.
* Кнопка «Проверить допуск».
* Место для результата.


## Задача 4. Скидка в магазине

**1. Назначение программы:**
Программа считает, сколько нужно заплатить за покупку с учётом скидки.

**2. Входные данные:**
* Сумма покупки в рублях: от 0 до 1 000 000.
* Статус клиента: «постоянный» или «новый».

**3. Как считается скидка:**
* Меньше 1000 руб. — скидки нет.
* От 1000 до 4999 руб. — скидка 5%.
* От 5000 руб. и выше — скидка 10%.
* Постоянным клиентам — ещё дополнительные 5% от итоговой суммы.

**4. Выходные данные:**
* Итоговая сумма с учётом скидки, округлённая до копеек.

**5. Как работает программа:**
1. Ввести сумму покупки.
2. Выбрать статус клиента.
3. Посчитать скидку по правилам выше.
4. Вычислить итог: S_итог = S * (1 - скидка).
5. Вывести результат.

**6. Что делать при ошибках:**
* Отрицательная сумма — «Ошибка: сумма не может быть отрицательной».
* Статус не выбран — «Выберите статус клиента».

**7. Интерфейс:**
* Поле для суммы.
* Выпадающий список для статуса.
* Кнопка «Рассчитать».
* Отображение скидки и итоговой суммы.


## Задача 5. Квадратное уравнение

**1. Назначение программы:**
Программа решает квадратное уравнение вида ax² + bx + c = 0.

**2. Входные данные:**
* Коэффициенты a, b, c — целые или дробные числа.
* a не может быть равно нулю, диапазон значений: от -1000 до 1000.

**3. Выходные данные:**
* Корни уравнения, если они есть.
* Или одно из сообщений:
    * «Уравнение не имеет действительных корней» — если D < 0,
    * «Один корень: x = ...» — если D = 0,
    * «Два корня: x1 = ..., x2 = ...» — если D > 0.

**4. Как работает программа:**
1. Ввести a, b, c.
2. Посчитать дискриминант: D = b² - 4ac.
3. В зависимости от D:
    * D < 0 → нет корней.
    * D = 0 → x = -b / (2a).
    * D > 0 → x1 = (-b + √D) / (2a), x2 = (-b - √D) / (2a).
4. Вывести результат.

**5. Что делать при ошибках:**
* a = 0 — «Ошибка: коэффициент a не может быть равен нулю».
* Не число — «Введите корректные числа».

**6. Интерфейс:**
* Три поля для коэффициентов.
* Кнопка «Решить».
* Место для вывода результата.
* Кнопка сброса полей.
