# Технические задания для программных модулей

## Задача 1 — Линейный алгоритм «Перевод времени»

**Назначение**

Программа для преобразования временных единиц из часов в минуты.

**Входные данные** — целое число, представляющее количество часов (≥ 0)

**Выходные данные** — целое число, представляющее эквивалент в минутах

**Алгоритм**

Получить значение часов от пользователя, умножить на 60, вывести результат.

**Формула**
```
минуты = часы × 60
```

**Пример**

Вход: `2 часа` → Выход: `120 минут`

## Задача 2 — Алгоритм ветвления «Сравнение чисел»

**Назначение**

Программа для сравнения двух чисел и определения их соотношения.

**Входные данные** — два числа: `a` и `b`

**Выходные данные** — `"a больше b"` / `"a меньше b"` / `"a равно b"`

**Алгоритм**

Получить два числа от пользователя. Если `a > b` — вывести `"a больше b"`. Если `a < b` — вывести `"a меньше b"`. Если `a = b` — вывести `"a равно b"`.

**Пример**

Вход: `a = 5, b = 3` → Выход: `"5 больше 3"`

## Задача 3 — Алгоритм ветвления «Допуск к экзамену»

**Назначение**

Программа для проверки допуска студента к экзамену на основе его текущей оценки.

**Входные данные** — текущая оценка студента (0–100 баллов)

**Выходные данные** — `"Допущен к экзамену"` или `"Не допущен к экзамену"`

**Условия допуска**

Оценка ≥ 60 баллов — допущен. Оценка < 60 баллов — не допущен.

**Алгоритм**

Получить текущую оценку студента. Если оценка ≥ 60 — вывести `"Допущен к экзамену"`, иначе — `"Не допущен к экзамену"`.

**Пример**

Вход: `75 баллов` → Выход: `"Допущен к экзамену"`

## Задача 4 — Алгоритм с ветвлением «Скидка в магазине»

**Назначение**

Программа для расчёта стоимости покупки с применением скидки в зависимости от суммы.

**Входные данные** — стоимость товара в рублях

**Выходные данные** — размер скидки (%), сумма скидки (руб.), итоговая стоимость (руб.)

**Условия скидки**

| Сумма покупки | Скидка |
|---|---|
| До 1 000 руб. | 0% |
| 1 000 — 5 000 руб. | 5% |
| 5 000 — 10 000 руб. | 10% |
| Свыше 10 000 руб. | 15% |

**Алгоритм**

Получить сумму покупки. Определить процент скидки по таблице. Вычислить размер скидки: `скидка = сумма × (процент / 100)`. Вычислить итог: `итог = сумма − скидка`. Вывести скидку и итоговую стоимость.

**Пример**

Вход: `7 500 руб.` → Скидка: `10% = 750 руб.` → Выход: `Итоговая стоимость = 6 750 руб.`

## Задача 5 — Алгоритм с вложенными ветвлениями «Квадратное уравнение»

**Назначение**

Программа для решения квадратного уравнения вида **ax² + bx + c = 0** и вывода корней или информации об их отсутствии.

**Входные данные** — коэффициенты `a` (a ≠ 0), `b` и `c`

**Выходные данные** — два корня, один корень, сообщение об отсутствии корней или сообщение об ошибке при `a = 0`

**Формулы**
```
D = b² − 4ac
x = (−b ± √D) / 2a
```

**Алгоритм**
```
1. Получить коэффициенты a, b, c
2. Если a = 0 → ошибка: "Это не квадратное уравнение"
3. Вычислить D = b² − 4ac
4. Если D > 0:
     x₁ = (−b + √D) / 2a
     x₂ = (−b − √D) / 2a
     → Вывести два корня
   Если D = 0:
     x = −b / 2a
     → Вывести один корень
   Если D < 0:
     → "Нет действительных корней"
```

**Таблица решений**

| Дискриминант | Количество корней | Действие |
|---|---|---|
| D > 0 | 2 | Вычислить оба корня |
| D = 0 | 1 | Вычислить один корень |
| D < 0 | 0 | Вывести сообщение об отсутствии |

**Примеры**

*D > 0* — Вход: `a=1, b=−5, c=6` → D = 1 → Выход: `x₁ = 3, x₂ = 2`

*D = 0* — Вход: `a=1, b=−2, c=1` → D = 0 → Выход: `x = 1`

*D < 0* — Вход: `a=1, b=0, c=1` → D = −4 → Выход: `"Нет действительных корней"`
